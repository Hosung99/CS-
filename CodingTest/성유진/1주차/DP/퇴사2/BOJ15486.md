### Problem
[퇴사2](https://www.acmicpc.net/problem/15486)


### Approach 
접근법1 -> 시간초과

가장 긴 감소하는 수열을 푼 방식과 동일하게 접근  
뒤에서부터 확인해서 더 큰 수가 등장하면 해당 위치에 횟수를 저장  
자신의 뒤에 있는 애들을 모두 확인해서 더 큰 횟수로 저장  


접근법2

```java
for (int i = N; i > 0; i--) {
  if (i + T[i] <= N) {
    dp[i] = Math.max(dp[i], P[i] + dp[i + T[i]]);
  } 
}
```
